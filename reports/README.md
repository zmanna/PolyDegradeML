# Reports

Generated artifacts are grouped by experiment stage.

| Folder | Contents |
|---|---|
| `week04/` | Dataset metadata and curation report artifacts. |
| `week05/` | Baseline-model metrics. |
| `week06/` | Feedforward neural network metrics. |
| `week07/` | Descriptor-graph prototype metrics. |
| `week08/` | Cross-environment validation metrics and charts. |
| `week09/` | Stratified validation diagnostics and model results. |
| `week10/` | Feature engineering diagnostics, results, and comparison chart. |
| `week11/` | Feature rankings, selected feature-set diagnostics, and charts. |
| `week12/` | Uncertainty metrics, prediction-level uncertainty, selective prediction, and charts. |
| `week13/` | Final model reliability scoreboard, reused uncertainty artifacts, and charts. |

Human-readable narrative summaries live in `docs/weekly/`.
