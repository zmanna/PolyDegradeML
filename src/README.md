# Source Code

This folder contains the reusable Python package for the project.

The package lives in:

```text
biodegradation_ml_framework/
```

Use `scripts/` for command-line workflows and `src/` for reusable data loading, modeling, validation, feature engineering, uncertainty, and reliability logic.

See `docs/uml/` for UML-style diagrams showing how the source modules connect to the scripts and generated outputs.
