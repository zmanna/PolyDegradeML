# Figures

This folder stores generated plots used in reports, presentations, and manuscript drafts.

- `cross_environment/`: proxy environment validation figures
- `feature_engineering/`: feature-set comparison figures
- `feature_importance/`: feature ranking and feature selection figures
- `uncertainty_calibration/`: calibration, uncertainty, and selective prediction figures
- `model_selection/`: final reliability scoreboard and model-selection figures

Figures are generated artifacts. Regenerate them by running the corresponding scripts in `scripts/`.
