# Result Tables

This folder stores generated CSV tables used for analysis, reports, and manuscript tables.

Key files:

- `stratified_cv_model_results.csv`
- `cross_environment_validation_summary.csv`
- `feature_importance_rankings.csv`
- `uncertainty_reliability_metrics.csv`
- `model_reliability_scoreboard.csv`

These files are regenerated by scripts in `scripts/`.

## Table Index

| Table | Supports | Generated By |
| --- | --- | --- |
| `stratified_cv_fold_diagnostics.csv` | Cross-validation split diagnostics | `scripts/06_run_stratified_cross_validation.py` |
| `stratified_cv_model_results.csv` | Standard model comparison and SMOTE comparison | `scripts/06_run_stratified_cross_validation.py` |
| `cross_environment_validation_summary.csv` | Distribution-shift/generalization results | `scripts/05_run_cross_environment_validation.py` |
| `feature_engineering_diagnostics.csv` | Feature-engineering validation diagnostics | `scripts/07_run_feature_engineering_comparison.py` |
| `feature_engineering_model_results.csv` | Baseline versus proxy feature comparison | `scripts/07_run_feature_engineering_comparison.py` |
| `feature_importance_rankings.csv` | Feature ranking and interpretation | `scripts/08_run_feature_importance_selection.py` |
| `feature_selection_model_results.csv` | Selected feature-set model comparison | `scripts/08_run_feature_importance_selection.py` |
| `feature_selection_cross_environment.csv` | Selected feature-set cross-environment results | `scripts/08_run_feature_importance_selection.py` |
| `uncertainty_reliability_metrics.csv` | Calibration and uncertainty reliability metrics | `scripts/09_run_uncertainty_reliability_analysis.py` |
| `cross_environment_uncertainty_metrics.csv` | Reliability behavior under distribution shift | `scripts/09_run_uncertainty_reliability_analysis.py` |
| `final_uncertainty_metrics.csv` | Reused final uncertainty metrics for model selection | `scripts/10_run_model_reliability_scoreboard.py` |
| `model_reliability_scoreboard.csv` | Final recommended model and reliability ranking | `scripts/10_run_model_reliability_scoreboard.py` |
