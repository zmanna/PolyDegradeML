# QSAR Biodegradation Descriptor Benchmark

This folder contains the current working dataset used by PolyDegradeML.

## Scope

This dataset is a descriptor-based QSAR/QSPR biodegradation benchmark. It is useful for testing tabular machine learning workflows, model comparison, feature engineering, uncertainty analysis, calibration, and reliability-aware model selection.

## What It Contains

- `1055` samples
- `41` numeric molecular descriptor columns
- Binary biodegradation labels:
  - `RB`: readily biodegradable
  - `NRB`: not readily biodegradable
- Curated sample identifiers for row-level traceability
- Machine-readable metadata describing curation choices and representation limits

## What It Does Not Contain

This is not a polymer-identity or structure-rich dataset. It does not include:

- polymer names
- chemical names
- repeat units
- SMILES strings
- BigSMILES strings
- atom-bond molecular graphs
- Morgan fingerprints
- environmental exposure metadata
- continuous degradation-rate constants

## Folder Contents

- `raw/`: source CSV as downloaded
- `processed/`: curated CSV generated by `scripts/01_curate_dataset.py`
- `metadata/`: dataset version, source path, curation decisions, labels, and limitations

## Scientific Use

Use this dataset to reproduce the current descriptor-based classification results. Do not use it to claim polymer-specific structure modeling, BigSMILES support, molecular graph learning, or degradation-rate prediction. Those capabilities require a different dataset with explicit polymer identity and structure fields.
