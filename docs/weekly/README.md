# Weekly Experiment Notes

This folder contains the human-readable experiment trail for the project.

The files are ordered by project stage:

| Stage | Focus |
|---|---|
| `week04-*` | Dataset curation, representation decisions, and source-data limitations. |
| `week05-*` | Baseline modeling and first classification results. |
| `week06-*` | Feedforward neural network experiment. |
| `week07-*` | Descriptor-graph prototype experiment. |
| `week08-*` | Cross-environment validation. |
| `week09-*` | Stratified validation and sampling comparison. |
| `week10-*` | Feature engineering comparison. |
| `week11-*` | Feature importance and feature-set selection. |
| `week12-*` | Uncertainty and reliability analysis. |
| `week13-*` | Final model-selection report and summary. |

Generated machine-readable artifacts live in `reports/weekXX/`.
