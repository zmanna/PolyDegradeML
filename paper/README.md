# Paper

This folder is for manuscript planning and paper-facing notes.

Current file:

- `outline.md`: working manuscript structure and mapping from project outputs to paper sections
- `evidence_mapped_outline.md`: manuscript outline mapped to repository evidence paths
- `manuscript_draft.md`: first scientific manuscript draft
- `tables_and_figures_plan.md`: proposed publication figures and tables
- `publication_readiness_checklist.md`: remaining checks before sharing or submission
- `missing_information_to_ask_author.md`: author/advisor questions needed for finalization

Use this folder to organize the scientific narrative. Keep reproducible computations in `src/` and `scripts/`; cite generated outputs from `results/`, `figures/`, and `reports/`.
