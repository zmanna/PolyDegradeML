# Data

This folder separates source inputs from curated modeling artifacts.

| Path | Purpose |
|---|---|
| `qsarbiodegradation/qsar-biodeg.csv` | Source descriptor dataset preserved as downloaded. |
| `processed/qsar_biodegradation_curated.csv` | Curated descriptor dataset used by the modeling scripts. |

The source dataset name references QSAR biodegradation because that is the upstream descriptor dataset label. In this repository, the dataset is used as the first modeling substrate for polymer degradation pathway prediction work.

Future open-source extensions should add dataset-specific documentation, licensing notes, and provenance details before adding new data files.
