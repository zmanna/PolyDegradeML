# Source Materials

This folder stores original project-development materials that support the scientific narrative but are not generated by the current reproducible workflow.

- `reports/`: weekly applied-work reports and outside-reading reports preserved as submitted source documents.

These materials provide provenance for the literature review, methodology evolution, and interpretation sections of the manuscript. Keep the imported documents intact; if summaries or citation matrices are needed, create new files alongside this README rather than editing the original reports.
