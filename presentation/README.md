# Presentation

This folder contains slide-deck materials for communicating the project.

- `PolyDegradeML_comprehensive_defense.pptx`: full publication-quality research presentation for conference, thesis-defense, or R&D review use
- `PolyDegradeML_comprehensive_defense_speaker_notes.md`: slide-by-slide speaker notes, evidence paths, cautions, and talk-track prompts for the comprehensive deck
- `build_comprehensive_defense_deck.mjs`: reproducible generator for the comprehensive deck
- `PolyDegradeML_academic_ted_talk.pptx`: shorter academic/TED-style communication deck
- `PolyDegradeML_academic_ted_talk_speaker_notes.md`: speaker notes for the shorter communication deck
- `build_academic_ted_talk.mjs`: reproducible generator for the shorter communication deck
- `PROJECT_SLIDES.md`: slide narrative and content outline
- `Plastic_Degradation_Project_Presentation.pptx`: generated or assembled PowerPoint deck
- `SLIDE_ASSET_GUIDE.md`: guide to slide assets
- `assets/`: images used in the presentation
- `build_project_slides.py`: slide-building helper script

Presentation materials summarize the research but are not the source of reproducible results.
The generated decks intentionally reference result files in `reports/`, `results/`, and `figures/`; regenerate the analysis outputs before formal submission if any upstream data or model scripts change.
