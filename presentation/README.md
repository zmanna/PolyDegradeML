# Presentation

This folder contains slide-deck materials for communicating the project.

- `PROJECT_SLIDES.md`: slide narrative and content outline
- `Plastic_Degradation_Project_Presentation.pptx`: generated or assembled PowerPoint deck
- `SLIDE_ASSET_GUIDE.md`: guide to slide assets
- `assets/`: images used in the presentation
- `build_project_slides.py`: slide-building helper script

Presentation materials summarize the research but are not the source of reproducible results.
